VERCEL ENVIRONMENT VARIABLES SETUP
==================================

To fix the DEPLOYMENT_NOT_FOUND error, you need to add these environment variables:

1. Go to: https://vercel.com/dashboard
2. Select your project: num1-store
3. Go to Settings -> Environment Variables
4. Add these variables:

Variable Name: NEXT_PUBLIC_SUPABASE_URL
Value: https://your-project-id.supabase.co
Environments: Production, Preview

Variable Name: NEXT_PUBLIC_SUPABASE_ANON_KEY
Value: your_supabase_anon_key
Environments: Production, Preview

After adding, redeploy with:
vercel --prod

Note: Get your Supabase credentials from:
- Your local .env.local file, OR
- Supabase Dashboard -> Settings -> API
